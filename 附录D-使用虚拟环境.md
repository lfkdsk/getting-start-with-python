## 附录D：使用虚拟环境

创建一个 Python 项目的时候，我们最好开启一个虚拟环境来安装程序所需要的第三方包。虚拟环境是相当于在系统中生成了一份 python 的备份，你可以在这个备份之中安装依赖包，并将其与其他的 Python 包隔离。
